简介

  • 旧版 Native 的 Flight status 年代久远，目前几乎没有继续迭代。其功能几乎完全独立且不涉及订单流程。故整体拆分出来交由 H5 团队，发挥 H5 灵活敏捷的优势，快速迭代新的玩法(如延误概率实时数据图表展示)
  • Tech stack
    • 又爱又恨的 Lizard (react-lite + requireJs...)
    • ES6 + Postcss
    • Hybrid API
    • 构建：gulp && 打包：webpack
